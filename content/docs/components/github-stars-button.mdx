## Features

- **Real-time Data**: Fetches star count from GitHub API on mount
- **Loading State**: Shows "..." while loading star count
- **Error Handling**: Gracefully handles fetch errors
- **Click to Repository**: Opens the GitHub repository in a new tab
- **Hover Effects**: Star icon turns yellow on hover
- **Responsive**: Works on all screen sizes
- **Lightweight**: No external dependencies except React and Radix Icons

## Props

| Prop      | Type                      | Default   | Description                    |
| --------- | ------------------------- | --------- | ------------------------------ |
| repo      | string                    | -         | GitHub repository (owner/repo) |
| showLabel | boolean                   | true      | Whether to show "GitHub" label |
| size      | "sm" \| "default" \| "lg" | "sm"      | Size of the button             |
| className | string                    | undefined | Additional CSS classes         |

## Notes

- The component uses GitHub's public API which has rate limiting (60 requests/hour for unauthenticated requests)
- Star count is fetched once on component mount
- The component automatically cleans up on unmount to prevent memory leaks
